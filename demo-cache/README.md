# Sample Reports

Pre-generated Afterburn reports from scanning the included FlowSync test site.

- **Health Score:** 62/100
- **Issues Found:** 33 (11 high, 12 medium, 10 low)
- **Mode:** Heuristic (no Gemini API key required)

## Files

- `report.html` — Human-readable HTML report with health score, prioritized issues, and fix suggestions
- `report.md` — AI-readable Markdown report for pasting into Claude or Cursor
- `terminal-output.txt` — CLI terminal output from the scan

## Fixed Site Comparison

The `fixed-site/` directory contains reports from a re-scan after fixing all issues — demonstrating the before/after workflow.
